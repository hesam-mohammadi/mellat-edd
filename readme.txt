== Description ==
درگاه پرداخت بانک ملت براي افزونه فروش فايل

* بدون تداخل با سایر افزونه ها و بدون نمایش ارور در بخش مدیریت
* به صورت خودکار به بخش درگاه های پرداخت اضافه می شود

Bank Mellat gateway for easy digital downloads (EDD)
== Installation ==
1. Upload "mellat-edd" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. In the payement gateways setting of the EDD plugin set the payment gateways and default gateways.
4. Complete the terminal id , username and password fields and click on save changes.
